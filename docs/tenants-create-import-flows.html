Flussi Funzione API Tenants

Endpoint analizzati: /api/v1/tenants/create (legacy) e /api/v1/tenants/provision-stream.

Flusso: /api/v1/tenants/create

flowchart TD
  reqCreate[POST tenants/create]
  reqCreate --> validateCreate{installationId userId slug ownerLogin validi?}
  validateCreate -- No --> createBadReq[400 ERR_BAD_REQUEST]
  validateCreate -- Yes --> slugNorm[Normalize slug + apiKey + load env Vercel]
  slugNorm --> vercelCfg{VERCEL_TEAM_ID e token presenti?}
  vercelCfg -- No --> createCfgErr[500 ERR_VERCEL_CONFIG]
  vercelCfg -- Yes --> ghStart[GitHub phase start]

  ghStart --> ghGenerate{createUsingTemplate ok?}
  ghGenerate -- Yes --> ghRepoOk[Repo creato da template]
  ghGenerate -- No --> ghFallback[Fallback create repo + copy blobs]
  ghFallback --> ghCopyOk[Tree + initial commit + ref branch]
  ghRepoOk --> ghDone[Save githubRepoId/fullName]
  ghCopyOk --> ghDone

  ghDone --> ghErrorCheck{GitHub error?}
  ghErrorCheck -- Name taken --> createConflict[409 ERR_GITHUB_NAME_TAKEN]
  ghErrorCheck -- Forbidden 403 --> createForbidden[403 ERR_GITHUB_TEMPLATE_FORBIDDEN]
  ghErrorCheck -- Other --> createGhFail[502 ERR_GITHUB_FAILED]
  ghErrorCheck -- No --> vercelProject[Vercel create project]

  vercelProject --> vercelProjectOk{Project creato?}
  vercelProjectOk -- Limit reached --> createLimit[402 ERR_VERCEL_LIMIT_REACHED]
  vercelProjectOk -- No --> createVercelFail[502 ERR_VERCEL_FAILED]
  vercelProjectOk -- Yes --> setEnv1[Set env VITE_JSONPAGES_CLOUD_URL + VITE_JSONPAGES_API_KEY]
  setEnv1 --> envOk{Env insert ok?}
  envOk -- No --> createEnvFail[502 ERR_VERCEL_ENV_FAILED]
  envOk -- Yes --> dbInsert[Insert tenant in Supabase status provisioned]
  dbInsert --> dbOk{Insert ok?}
  dbOk -- No --> createDbFail[500 ERR_SUPABASE_FAILED]
  dbOk -- Yes --> setTenantEnv[Try set JSONPAGES_TENANT_ID non-fatal]
  setTenantEnv --> createSuccess[200 success + tenant + api_key]
