Login Flow

Action-Oriented Function Flow from Home click Login to Cloud to Dashboard render.

Flowchart

flowchart TD
  userClick[User clicks LoginToCloud] --> loginBtn[LoginButton handleLogin]
  loginBtn --> oauthStart[Supabase signInWithOAuth provider github]
  oauthStart --> setRedirect[Set redirectTo /dashboard]
  setRedirect --> githubAuth[Browser goes to GitHub OAuth consent]
  githubAuth --> callback[Supabase completes auth callback]
  callback --> goDashboard[Browser lands on /dashboard]

  goDashboard --> dashInit[DashboardPage useEffect init]
  dashInit --> getUser[supabase.auth.getUser]
  getUser --> hasUser{User exists}
  hasUser -- no --> backHome[router.push /]
  hasUser -- yes --> loadProjects[Query tenants from Supabase]
  loadProjects --> renderDash[Render Projects dashboard]

Sequence Diagram

sequenceDiagram
  participant User
  participant Home as HomePage
  participant Login as LoginButton
  participant SupabaseAuth as SupabaseAuth
  participant GitHub as GitHubOAuth
  participant Dashboard as DashboardPage
  participant SupabaseDB as SupabaseDB

  User->>Home: Click LoginToCloud
  Home->>Login: onClick handleLogin()
  Login->>SupabaseAuth: signInWithOAuth(github, redirectTo=/dashboard)
  SupabaseAuth->>GitHub: Redirect to OAuth consent
  GitHub-->>SupabaseAuth: OAuth success callback
  SupabaseAuth-->>User: Redirect browser to /dashboard

  User->>Dashboard: Open /dashboard
  Dashboard->>SupabaseAuth: auth.getUser()
  alt user missing
    Dashboard-->>User: Redirect to /
  else user valid
    Dashboard->>SupabaseDB: select tenants order by created_at desc
    SupabaseDB-->>Dashboard: projects list
    Dashboard-->>User: Show Projects UI
  end
